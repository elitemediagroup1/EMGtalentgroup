# EMG Talent Group — Premium Editorial Redesign V2

This package is a full replacement for the existing static site.

## Deployment

1. Back up the current GitHub repository.
2. Replace the repository files with the contents of this folder.
3. Commit to a preview branch first.
4. Review the Netlify deploy preview.
5. Merge only after verifying the three Netlify forms.

## Preserved form names

- `creator-application`
- `talent-access-request`
- `general-and-brand-inquiry`

Form email notifications remain configured in Netlify and should continue routing to `hello@elitemediagroup.io`.

## Visual changes

- Full-bleed photographic heroes
- Editorial typography and layouts
- Realistic photography across all creator niches
- Distinct page identities
- Product-style Creator Hub presentation
- Media-portfolio treatment for the In My City ecosystem
- New navigation, footer, responsive behavior, and accessible dropdowns
- Correct page-specific canonical URLs and noindex handling for utility pages

Photography is loaded from optimized Unsplash CDN URLs. Stock subjects are used only as editorial category imagery and are not represented as signed EMG creators.
